UTAPS Website

A website created for the University Transport and Parking Services. The purpose of the website is to help show student's alternative commute options in order to improve transportation and parking at the university. Major technologies used are HTML, CSS, Node.js, and Google Maps API.

Pages

Welcome Page - Provides a quick summary to the user about the tool and its' purpose.


Survey Page - Gather information about the user to gain better insight of the communte practices of the user.


Transit Page - User selects a communte option they are intrested in testing the results of cost, enviormental impact, and travel time.


Routing Page - The user inputs the route they would like to take using the method of transport they chose before and will be used to make the final calculations.


Results Page - Provides visulaizatoin of costs, enviormental impact, and travel time. They exact numbers are also shown below.
